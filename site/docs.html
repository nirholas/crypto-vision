Installation

Get XTools up and running in minutes.

Requirements

  • Python 3.10 or higher
  • pip (Python package manager)
  • A modern browser (Chromium recommended)

Install via pip

# Install XTools
pip install xtools

# Install Playwright browser
playwright install chromium

Install from source

# Clone the repository
git clone https://github.com/nirholas/Get-Tweet-Replies-With-Python-Tweepy.git
cd Get-Tweet-Replies-With-Python-Tweepy/xtools

# Install in development mode
pip install -e .

# Install browser
playwright install chromium

Verify Installation

# Check version
xtools --version

# Run health check
xtools doctor
Note: XTools uses Playwright for browser automation. The first run may take longer as it downloads browser binaries.

Quick Start

Your first XTools automation in 5 minutes.

Step 1: Login

# Interactive login (opens browser window)
xtools auth login

# This saves your session for future use

Step 2: Your First Script

import asyncio
from xtools import XTools

async def main():
    async with XTools() as x:
        # Like 10 tweets about Python
        results = await x.engage.like_by_keyword(
            keyword="Python programming",
            limit=10
        )
        
        print(f"Liked {len(results)} tweets!")
        
        for result in results:
            print(f"  ✓ @{result.author_username}: {result.tweet_text[:50]}...")

asyncio.run(main())

Step 3: Run It

python my_first_script.py
Tip: Always start with dry_run=True to preview actions without actually performing them.

Authentication

XTools uses browser sessions - no API keys required.

Interactive Login

The easiest way to authenticate:

# Opens a browser window for login
xtools auth login

# Your session is saved to ~/.xtools/session.json

Import Existing Cookies

If you have cookies exported from your browser:

from xtools import XTools

async with XTools() as x:
    # Import from JSON file (e.g., from browser extension)
    await x.auth.import_cookies("twitter_cookies.json")
    
    # Import from Netscape format
    await x.auth.import_cookies("cookies.txt", format="netscape")

Programmatic Login

from xtools import XTools

async with XTools(headless=False) as x:
    # Manual login with timeout
    await x.auth.login(timeout=120)  # 2 minutes to complete login
    
    # Save session for reuse
    await x.auth.save_session("my_session.json")

Load Saved Session

from xtools import XTools

async with XTools() as x:
    # Load previously saved session
    await x.auth.load_session("my_session.json")
    
    # Verify authentication
    if await x.auth.is_authenticated():
        print("✓ Logged in!")
    else:
        print("✗ Session expired")
Security: Never share your session files. They contain authentication tokens that grant full access to your account.

Like by Keyword

from xtools import XTools
from xtools.actions.types import LikeFilters

# Set up filters
filters = LikeFilters(
    min_likes=10,           # Skip low-engagement tweets
    min_followers=500,      # Quality threshold
    max_followers=100000,   # Skip mega-accounts
    exclude_retweets=True,  # Only original content
    exclude_replies=False,  # Include replies
    language="en",          # English only
    exclude_keywords=["spam", "giveaway", "follow4follow"]
)

async with XTools() as x:
    results = await x.engage.like_by_keyword(
        keyword="machine learning",
        limit=25,
        filters=filters
    )
    
    print(f"Liked {len([r for r in results if r.success])} tweets")

Auto-Liker (Continuous)

from xtools import XTools
from xtools.actions.types import AutoLikeConfig, LikeFilters

config = AutoLikeConfig(
    keywords=["Python", "AI", "startups"],
    max_likes_per_hour=30,      # Stay within safe limits
    min_delay=45.0,              # Minimum seconds between actions
    max_delay=180.0,             # Maximum seconds between actions
    skip_retweets=True,
    engagement_threshold=50,     # Min total engagement
)

filters = LikeFilters(
    min_likes=20,
    min_followers=1000,
)

async with XTools() as x:
    liker = x.engage.create_auto_liker(config, filters)
    
    # Run for 2 hours
    async for result in liker.run(duration_hours=2):
        if result.success:
            print(f"✓ Liked @{result.author_username}")
        
        # Check stats
        stats = liker.get_stats()
        print(f"  Total: {stats.total_liked} | Rate: {stats.likes_per_hour}/hr")

Bot Detection

from xtools.ai import BotDetector

detector = BotDetector()

async with XTools() as x:
    followers = await x.scrape.followers("me", limit=500)
    
    bots = []
    for follower in followers:
        score = await detector.analyze(follower)
        
        if score.bot_probability > 0.8:
            bots.append({
                "username": follower.username,
                "probability": score.bot_probability,
                "reasons": score.reasons
            })
    
    print(f"Found {len(bots)} likely bots")
    
    # Reasons might include:
    # - "Default profile picture"
    # - "Account created recently"
    # - "No original tweets"
    # - "Follows/follower ratio > 10"

Best Practices

Account Safety

✓ Do

  • Start slow - begin with conservative rate limits
  • Use dry_run mode to test new automations
  • Maintain a healthy following/follower ratio
  • Engage with relevant content in your niche
  • Use human-like delays (random, not fixed)
  • Take breaks - don't run 24/7

✗ Don't

  • Run aggressive automation on new accounts
  • Like/follow hundreds of accounts per day
  • Use identical comments repeatedly
  • Ignore rate limit warnings
  • Automate controversial or spam content

Engagement Quality

  • Be selective: Use filters to engage with quality content only
  • Be relevant: Comment templates should match the tweet's context
  • Be genuine: AI comments should add value, not just exist
  • Be patient: Organic growth takes time, even with automation
